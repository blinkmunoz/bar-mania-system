# Bar Mania System

Sistema de controle de estoque e comandas desenvolvido com React.

## Funcionalidades

- Cadastro de produtos
- Controle de estoque
- Comandas de mesas
- Baixa automática do estoque
- Busca de produtos
- Dashboard administrativo

## Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS3

## Como executar o projeto

```bash
npm install
npm run dev
```
